Jedes Interview folgt diesem Format:

```markdown
# Interview: [Stakeholder Name]

**Date:** YYYY-MM-DD  
**Interviewer:** [Name]  
**Duration:** [X min]  
**Role:** [Stakeholder Role]

## Key Takeaways

- Insight 1
- Insight 2
- Insight 3

## Quotes

> "Zitat vom Stakeholder..."

## Requirements Gathered

- REQ-001: [Requirement]
- REQ-002: [Requirement]

## Next Steps

- [ ] Action Item 1
- [ ] Action Item 2

## Artifacts

- Fotos/Sketches: [Links]
```
