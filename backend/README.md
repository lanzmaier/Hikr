# Hikr Backend

## Build System

This project uses **Maven** as its build system.

## Dependencies

The following Spring Boot dependencies are used in this project:

- **Spring Web** - For building RESTful web services and handling HTTP requests
- **Spring Data JPA** - For database access and persistence layer management
- **Spring Security** - For authentication and authorization

## Getting Started

To build and run the project:

```bash
# Navigate to the hikr-backend directory
cd hikr-backend

# Build the project
./mvnw clean install

# Run the application
./mvnw spring-boot:run
```
