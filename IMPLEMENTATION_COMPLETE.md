## 🚀 How to Get Started

### Für ein NEW TEAM MEMBER (30 Minuten)
```
1. Lese: docs/ONBOARDING.md
   → Workspace Setup Instructions
   → Team-Spezifische Guides
   → FAQ & Troubleshooting

2. Setup Workspace:
   → Clone Repository
   → Öffne Workspace
   → Installiere Dependencies

3. Lese Relevant Agent:
   → Backend, Web, Mobile, oder PM Agent

4. Starte mit First Task:
   → Nutze Agent-Support
   → Follow Validation Checklist
   → Commit & PR erstellen
```

### Für ein ETABLIERTES TEAM
```
1. Review: docs/AI_RESOURCES_INDEX.md
   → Quick Navigation zu allen Resources

2. Per Task:
   → Öffne relevanten Agent
   → Kopiere Workflow/Prompt
   → Nutze in GitHub Copilot Chat
   → Follow Validation Checklist

3. Quality Gates:
   → Use Issue/PR Templates
   → Check Coding Standards
   → Run Tests & Linting
   → Submit for Review
```

## 🔧 Technische Details

### Tech Stack
- **Backend:** Java 21, Spring Boot 3.x, Maven
- **Web:** React 18+, TypeScript, Vite
- **Mobile:** React Native, Expo
- **API:** OpenAPI 3.0
- **Design:** Tailwind CSS + Design Tokens
- **Testing:** JUnit 5, Vitest, RN Testing Library

### Architecture Patterns
- **Backend:** Domain-Driven Design (DDD)
- **Frontend:** Component-Driven Design (CDD)
- **Repository:** Mono-Repository with Multi-Root Workspaces
- **API-First:** OpenAPI 3.0 Specification

### Quality Tools
- **Linting:** ESLint, Checkstyle
- **Formatting:** Prettier, Spotless
- **Testing:** JUnit 5, Vitest, React Testing Library
- **CI/CD:** GitHub Actions
